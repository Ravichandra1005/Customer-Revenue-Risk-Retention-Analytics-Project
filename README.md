**🚀 Customer Revenue, Risk & Retention Analytics Project**

**📌 Overview**
Analyzed customer behavior, revenue trends, returns, and delivery performance using Python and Pandas to generate business insights.

**📂 Data**
Converted Excel sheets → CSV files
customers.csv, orders.csv, order_items.csv
Merged using customer_id and order_id

**⚙️ Processing**
Created features:
order_value, delivery_delay_days
is_repeat_customer, is_high_value_order
customer_value_segment, customer_risk_category
order_reliability_score

**❓ Key Insights**
High revenue customers also have high returns → ⚠️ Risk
Cities like Mumbai & Chennai → ⏱️ Delivery delays
Repeat customers → 💰 More valuable
Clothing & Electronics → 🔁 High returns
UPI → 🚨 Highest risk payment mode

📊** Visualizations**
📈 Monthly revenue trend
📊 Customer segmentation
📊 Return rate by category
🔵 Delay vs revenue
🔥 City vs category heatmap

**🚨 Problems**
High return rates from valuable customers
Delivery delays in key cities
High-return product categories

**💡 Recommendations**
Improve product quality & descriptions
Optimize delivery in delay-prone cities
Reduce returns in Clothing & Electronics

**🛠️ Tools**
Python • Pandas • NumPy • Matplotlib • Seaborn

**⭐ Summary**
End-to-end data analytics project covering data cleaning, feature engineering, analysis, and visualization.
